# Uma simples lista de todo escrita em typescript puro

## Iniciando

### Instale as dependências globais
`Use sudo no linux/macOS`

```
npm install -g typescript parcel jest
```

### Instale as dependências locais

#### Na pasta do projeto digite:

```
npm i
```

### Inicie o projeto


#### Na pasta do projeto digite:

```
npm run start
```
